## Установка
Установка зависимостей
```sh
$ npm install
```

## Разработка
Запуск локального сервера webpack-dev-server [http://localhost:8080/](http://localhost:8080/)
```sh
$ npm run dev
```

## Деплой
```sh
$ npm run build
```
